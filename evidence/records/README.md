# Evidence Records

This tree holds the per-record v3.1 machine bundles.

Each record folder can contain:

- `metadata.json`
- `runtime.json`
- `static.json`
- `behavior.json`
- `classification.json`
- `timeline.json`
- `verdict.md`
- `full-evidence.json`
- `re-audit.json`

Lane manifests such as `runtime-lane.json`, `procmon-lane.json`, and `behavior-lane.json` also live here when a wrapper runs.
